##Arduino Heartbeat Monitoring System

## 📌 About the Project
This project is an IoT-based system using Arduino and a heartbeat sensor to monitor pulse rate.

---

## 🛠️ Components Used
- Arduino Uno
- Heartbeat Sensor
- Jumper Wires
- Breadboard

---

## ⚙️ Working
The sensor detects heartbeat signals and sends data to Arduino, which processes and displays it.
